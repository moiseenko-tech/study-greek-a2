# Частицы и служебные слова

## Утвердительные

| Слово | Перевод | Пример |
|-------|---------|--------|
| ναι | да | Ναι, θέλω καφέ. — Да, хочу кофе. |
| βέβαια | конечно | Βέβαια, μπορώ να βοηθήσω. — Конечно, могу помочь. |
| μάλιστα | да (формально), именно | Μάλιστα, κύριε. — Да, господин. |
| φυσικά | естественно | Φυσικά, έρχομαι! — Естественно, иду! |

## Отрицательные

| Слово | Перевод | Пример |
|-------|---------|--------|
| δεν | не (с изъявительным наклонением) | Δεν καταλαβαίνω. — Я не понимаю. |
| μην | не (с повелительным и сослагательным) | Μην μιλάς! — Не говори! |
| όχι | нет | Όχι, δεν θέλω. — Нет, не хочу. |
| κανείς / καμία | никто / никакой | Κανείς δεν ήρθε. — Никто не пришёл. |
| πουθενά | нигде, никуда | Δεν πάω πουθενά. — Я никуда не иду. |

## Неопределённость

| Слово | Перевод | Пример |
|-------|---------|--------|
| κάτι | что-то, что-нибудь | Θέλω κάτι να φάω. — Хочу что-нибудь поесть. |
| κάποιος / κάποια / κάποιο | кто-то, какой-то | Κάποιος τηλεφώνησε. — Кто-то позвонил. |
| κάπου | где-то, где-нибудь | Το άφησα κάπου. — Я оставил это где-то. |
| κάποτε | когда-то, иногда | Κάποτε πήγαινα εκεί. — Когда-то я туда ходил. |
| κάπως | как-то, как-нибудь | Κάπως θα τα καταφέρουμε. — Как-нибудь справимся. |
| μερικοί / μερικές / μερικά | некоторые, несколько | Μερικοί άνθρωποι δεν καταλαβαίνουν. — Некоторые люди не понимают. |
| λίγο | немного, чуть-чуть | Θέλω λίγο νερό. — Хочу немного воды. |
| κάθε | каждый, каждая | Κάθε μέρα πηγαίνω στη δουλειά. — Каждый день хожу на работу. |

## Вопросительные

| Слово | Перевод | Пример |
|-------|---------|--------|
| τι | что | Τι θέλεις; — Что ты хочешь? |
| ποιος / ποια / ποιο | кто, какой | Ποιος είναι αυτός; — Кто это? |
| πού | где, куда | Πού είναι το βιβλίο; — Где книга? |
| πότε | когда | Πότε φτάνεις; — Когда ты приедешь? |
| πώς | как | Πώς είσαι; — Как ты? |
| γιατί | почему | Γιατί αργείς; — Почему ты опаздываешь? |
| πόσος / πόση / πόσο | сколько, насколько | Πόσο κάνει; — Сколько стоит? |

## Причина и следствие

| Слово | Перевод | Пример |
|-------|---------|--------|
| γιατί | потому что | Μένω σπίτι γιατί βρέχει. — Остаюсь дома, потому что идёт дождь. |
| επειδή | потому что (более формально) | Δεν ήρθα επειδή ήμουν άρρωστος. — Не пришёл, потому что был болен. |
| γι' αυτό | поэтому | Βρέχει, γι' αυτό μένω σπίτι. — Идёт дождь, поэтому остаюсь дома. |
| άρα | следовательно, значит | Δεν απάντησε, άρα δεν ήξερε. — Не ответил, значит не знал. |
| λοιπόν | итак, ну что ж | Λοιπόν, πάμε; — Ну что, пошли? |

## Место и направление

| Слово | Перевод | Пример |
|-------|---------|--------|
| εδώ | здесь, сюда | Έλα εδώ! — Иди сюда! |
| εκεί | там, туда | Το κλειδί είναι εκεί. — Ключ там. |
| πάνω | вверху, наверх | Ανέβα πάνω. — Поднимись наверх. |
| κάτω | внизу, вниз | Το πορτοφόλι έπεσε κάτω. — Кошелёк упал на пол. |
| μέσα | внутри, внутрь | Μπες μέσα. — Заходи. |
| έξω | снаружи, на улице | Πήγαινε έξω. — Иди на улицу. |
| δίπλα | рядом | Κάθισε δίπλα μου. — Сядь рядом со мной. |
| μπροστά | впереди, перед | Στάσου μπροστά. — Встань впереди. |
| πίσω | сзади, за | Κάθεται πίσω. — Он сидит сзади. |

## Время

| Слово | Перевод | Пример |
|-------|---------|--------|
| τώρα | сейчас | Τώρα δεν μπορώ. — Сейчас не могу. |
| πάντα | всегда | Πάντα αργεί. — Он всегда опаздывает. |
| ποτέ | никогда (с δεν), когда-либо | Δεν πάω ποτέ εκεί. — Я никогда туда не хожу. |
| ήδη | уже | Έχω ήδη φάει. — Я уже поел. |
| ακόμα / ακόμη | ещё, всё ещё | Ακόμα κοιμάται. — Он ещё спит. |
| πια | больше (не), уже | Δεν τον βλέπω πια. — Я его больше не вижу. |
| σύντομα | скоро | Θα έρθω σύντομα. — Скоро приду. |
| αμέσως | сразу, немедленно | Έλα αμέσως! — Иди сразу! |
| μετά | потом, после | Θα μιλήσουμε μετά. — Поговорим потом. |
| πριν | до, раньше | Ήρθα πριν από σένα. — Я пришёл раньше тебя. |

## Контекстные замены

Одно русское слово переводится по-разному в зависимости от типа предложения.
В вопросах и отрицаниях используются одни и те же греческие слова.

| Русский | Утверждение | Вопрос | Отрицание (с δεν) |
|---------|-------------|--------|-------------------|
| возможно | ίσως | μήπως | — |
| кто-то / кто-нибудь / никто | κάποιος | κανείς | κανείς |
| что-то / что-нибудь / ничего | κάτι | τίποτα | τίποτα |
| где-то / где-нибудь / нигде | κάπου | πουθενά | πουθενά |
| когда-то / когда-нибудь / никогда | κάποτε | ποτέ | ποτέ |

### Примеры

| | Греческий | Русский |
|-|-----------|---------|
| утверждение | Κάποιος τηλεφώνησε. | Кто-то позвонил. |
| вопрос | Τηλεφώνησε κανείς; | Кто-нибудь позвонил? |
| отрицание | Δεν τηλεφώνησε κανείς. | Никто не позвонил. |
| | | |
| утверждение | Ίσως έρθει αύριο. | Возможно, придёт завтра. |
| вопрос | Μήπως ξέρεις πού είναι; | Возможно, ты знаешь, где это? |
| | | |
| утверждение | Θέλω κάτι να φάω. | Хочу что-нибудь поесть. |
| вопрос | Θέλεις τίποτα; | Хочешь что-нибудь? |
| отрицание | Δεν θέλω τίποτα. | Не хочу ничего. |

## Союзы и модальные слова

| Слово | Перевод | Пример |
|-------|---------|--------|
| και | и | Εγώ και εσύ. — Я и ты. |
| αλλά | но | Θέλω αλλά δεν μπορώ. — Хочу, но не могу. |
| όμως | однако | Είναι δύσκολο, όμως δεν είναι αδύνατο. — Трудно, однако не невозможно. |
| ή | или | Καφές ή τσάι; — Кофе или чай? |
| αν | если | Αν θέλεις, έλα. — Если хочешь, приходи. |
| ίσως | может быть | Ίσως έρθω αύριο. — Может, приду завтра. |
| μάλλον | наверное, скорее | Μάλλον δεν έρχεται. — Наверное, не придёт. |
| μόνο | только | Θέλω μόνο νερό. — Хочу только воду. |
| επίσης | также, тоже | Εγώ επίσης. — Я тоже. |
| ακριβώς | именно, точно | Ακριβώς αυτό ήθελα. — Именно это я и хотел. |
