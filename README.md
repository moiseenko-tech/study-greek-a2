# Греческий язык A2

Материалы для изучения новогреческого языка на уровне A2.

## Содержание

- [Глаголы](VERBS.md) — список глаголов по темам со спряжениями (настоящее, прошедшее, будущее время)
- [Прилагательные](ADJECTIVES.md) — размер, цвет, качество, состояние, характер, количество
- [Профессии и занятия](PROFESSIONS.md) — м./ж. формы и характерные действия по категориям
- [Числительные](NUMERALS.md) — количественные, склонение, порядковые, употребление
- [Время и даты](TIME_AND_DATES.md) — время суток, часы, дни недели, месяцы, сезоны, даты, возраст
- [Частицы и служебные слова](PARTICLES.md) — утвердительные, отрицательные, вопросительные, наречия места и времени, союзы
