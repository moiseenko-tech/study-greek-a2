## Количественные: тысячи

| # | Муж. | Жен. | Ср. | Перевод |
|---|------|------|-----|---------|
| 1 000 | χίλιοι | χίλιες | χίλια | тысяча |
| 2 000 | δύο χιλιάδες | δύο χιλιάδες | δύο χιλιάδες | две тысячи |
| 1 000 000 | ένα εκατομμύριο | — | — | миллион |

Для 1000 используется форма, согласующаяся с родом существительного.
Для 2000 и выше — слово `χιλιάδες` (существительное жен. рода) остаётся неизменным.

## Составные числительные

Компоненты пишутся раздельно. Изменяемые части (1, 3, 4 и сотни) согласуются с существительным:

| Число | Пример |
|-------|--------|
| 21 | είκοσι ένας / μία / ένα |
| 35 | τριάντα πέντε |
| 213 | διακόσιοι δεκατρείς (м.) / διακόσιες δεκατρείς (ж.) / διακόσια δεκατρία (с.) |
| 1 001 | χίλιοι ένας (м.) / χίλιες μία (ж.) / χίλια ένα (с.) |

## Особенности склонения

### ένας / μία / ένα — склоняется по падежам

Единственное числительное с полным склонением по падежам (совпадает с неопределённым артиклем).

| Падеж | Муж. | Жен. | Ср. |
|-------|------|------|-----|
| Именительный | ένας | μία | ένα |
| Винительный | έναν / ένα | μία | ένα |
| Родительный | ενός | μιας | ενός |

### τρεις / τρία и τέσσερις / τέσσερα — две формы рода

Мужской и женский род совпадают; средний отличается.

| Падеж | Муж./Жен. | Ср. |
|-------|-----------|-----|
| Им./Вин. | τρεις | τρία |
| Родительный | τριών | τριών |

| Падеж | Муж./Жен. | Ср. |
|-------|-----------|-----|
| Им./Вин. | τέσσερις | τέσσερα |
| Родительный | τεσσάρων | τεσσάρων |

### δύο — несклоняемое

Не меняется ни по роду, ни по падежу: δύο во всех формах.

### Сотни и χίλιοι — согласование с родом существительного

| | Пример | Перевод |
|-|--------|---------|
| Муж. | διακόσιοι άντρες | двести мужчин |
| Жен. | διακόσιες γυναίκες | двести женщин |
| Ср. | διακόσια παιδιά | двести детей |
| | | |
| Муж. | χίλιοι στρατιώτες | тысяча солдат |
| Жен. | χίλιες λέξεις | тысяча слов |
| Ср. | χίλια ευρώ | тысяча евро |
