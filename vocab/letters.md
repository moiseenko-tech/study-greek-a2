# Письма и переписка — Γράμματα και αλληλογραφία

## Официальное письмо (в газету, организацию, редакцию)

### Обращение

| Греческий | Перевод |
|-----------|---------|
| Αξιότιμε κύριε / Αξιότιμη κυρία | Уважаемый господин / Уважаемая госпожа |
| Αξιότιμοι κύριοι | Уважаемые господа |
| Προς τη σύνταξη | В редакцию |
| Προς τον/την υπεύθυνο/η | Ответственному лицу |

### Вступление

| Греческий | Перевод |
|-----------|---------|
| Σας γράφω σχετικά με... | Пишу Вам по поводу... |
| Σας αποστέλλω... | Направляю Вам... |
| Θα ήθελα να σας προτείνω... | Я хотел бы предложить Вам... |
| Αναφορικά με... | Относительно... / В связи с... |
| Επικοινωνώ μαζί σας για να... | Обращаюсь к Вам, чтобы... |
| Σας γράφω για να εκφράσω... | Пишу Вам, чтобы выразить... |

### Развитие темы

| Греческий | Перевод |
|-----------|---------|
| Κατά τη γνώμη μου... | По моему мнению... |
| Πιστεύω ότι... | Я считаю, что... |
| Είμαι της γνώμης ότι... | Я придерживаюсь мнения, что... |
| Θεωρώ ότι... | Я полагаю, что... |
| Επιπλέον... | Кроме того... / Вдобавок... |
| Ωστόσο... | Однако... |
| Από τη μια πλευρά... / Από την άλλη... | С одной стороны... / С другой... |
| Για παράδειγμα... | Например... |
| Συγκεκριμένα... | В частности... / Конкретно... |

### Завершение

| Греческий | Перевод |
|-----------|---------|
| Ελπίζω η απάντησή σας να είναι θετική. | Надеюсь на положительный ответ. |
| Είμαι στη διάθεσή σας για οποιαδήποτε πληροφορία. | Готов предоставить любую информацию. |
| Σας ευχαριστώ για τον χρόνο σας. | Благодарю за уделённое время. |
| Αναμένω την απάντησή σας. | Жду Вашего ответа. |

### Подпись

| Греческий | Перевод |
|-----------|---------|
| Με εκτίμηση, | С уважением, |
| Με τιμή, | С почтением, |
| Ευχαριστώ, | Благодарю, |

## Неформальное письмо другу

### Обращение

| Греческий | Перевод |
|-----------|---------|
| Αγαπητέ/ή [όνομα], | Дорогой/ая [имя], |
| Γεια σου [όνομα]! | Привет, [имя]! |
| Φίλε/Φίλη μου, | Мой друг / Моя подруга, |

### Вступление

| Греческий | Перевод |
|-----------|---------|
| Πώς είσαι; | Как ты? |
| Ελπίζω να είσαι καλά. | Надеюсь, ты в порядке. |
| Πολύς καιρός που δεν μιλήσαμε! | Давно не общались! |
| Σου γράφω γιατί... | Пишу тебе, потому что... |
| Θέλω να σου πω ότι... | Хочу тебе сказать, что... |
| Τι κάνεις τελευταία; | Что у тебя нового? |

### Завершение

| Греческий | Перевод |
|-----------|---------|
| Γράψε μου σύντομα! | Пиши мне скорее! |
| Ανυπομονώ να σε δω. | Не терпится тебя увидеть. |
| Πάρε με τηλέφωνο. | Позвони мне. |
| Σε φιλώ. | Целую тебя. |
| Φιλιά! | Поцелуи! |
| Τα λέμε! | Пока! / Увидимся! |

## Электронное письмо (email)

### Тема и структура

| Греческий | Перевод |
|-----------|---------|
| Θέμα: | Тема: |
| Σε απάντηση του email σου / σας... | В ответ на ваше письмо... |
| Σας αποστέλλω συνημμένο... | Направляю вам во вложении... |
| Παρακαλώ βρείτε συνημμένο... | Пожалуйста, см. вложение... |
| Σας ευχαριστώ για το email σας. | Благодарю за ваше письмо. |

## Рецензия в газету — полезные фразы

| Греческий | Перевод |
|-----------|---------|
| Θα ήθελα να σας στείλω μια κριτική για... | Хотел бы прислать вам рецензию на... |
| Παρακολούθησα πρόσφατα την ταινία... | Недавно я посмотрел фильм... |
| Θεωρώ ότι αξίζει να προβληθεί... | Считаю, что он заслуживает внимания... |
| Θα χαρώ αν δημοσιεύσετε αυτό το κείμενο. | Буду рад, если вы опубликуете этот текст. |
| Είμαι αναγνώστης/τρια της εφημερίδας σας. | Я читатель вашей газеты. |
